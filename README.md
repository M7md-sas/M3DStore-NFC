# M3DSTORE — صفحة طلب بطاقات NFC

موقع ثابت (صفحة واحدة) لعرض خدمة طباعة وتصميم بطاقات الأعمال الذكية NFC من نوع PVC، يحوّل طلبات العملاء مباشرة لواتساب.

## هيكل الملفات

```
/
├── index.html              الصفحة الرئيسية (كل المحتوى والنصوص)
├── assets/
│   ├── css/
│   │   └── styles.css      كل التنسيقات والألوان
│   ├── js/
│   │   └── script.js       منطق النموذج وتحويل واتساب
│   └── images/
│       ├── logo.png        ← ضع شعار البراند هنا
│       ├── card-programmed.jpg
│       ├── card-print.jpg
│       └── card-design.jpg
├── .nojekyll               يمنع GitHub Pages من معالجة Jekyll
├── .gitignore
└── README.md
```

> كل نوع ملف في مجلده: المحتوى في `index.html`، الشكل في `assets/css`، السلوك في `assets/js`، والصور في `assets/images`.

## كيف أعدّل (دليل سريع)

| أبغى أعدّل... | الملف | المكان |
|---|---|---|
| الأسعار / نصوص الباقات | [index.html](index.html) | قسم `<section class="section packages">` **+** قائمة `<select id="serviceSelect">` |
| رقم الواتساب | [assets/js/script.js](assets/js/script.js) | المتغيّر `WHATSAPP_NUMBER` في أعلى الملف |
| روابط الواتساب/الإنستقرام في الأسفل | [index.html](index.html) | قسم `<footer>` |
| الألوان والهوية | [assets/css/styles.css](assets/css/styles.css) | المتغيّرات في `:root` بأعلى الملف |
| الشعار | استبدل ملف `assets/images/logo.png` | (نفس الاسم) |
| صور البطاقات | استبدل ملفات `assets/images/card-*.jpg` | (نفس الأسماء) |

## الصور

صور البطاقات في [assets/images/](assets/images/) من **Unsplash** (ترخيص مجاني للاستخدام التجاري، بدون إلزام بالإسناد):
- `card-programmed.jpg` — بطاقات بيضاء بدون طباعة (النوع الأول)
- `card-print.jpg` — بطاقات بشعارات جاهزة (النوع الثاني)
- `card-design.jpg` — بطاقة مصممة احترافياً (النوع الثالث)

تقدر تستبدل أيّاً منها بصور منتجاتك الحقيقية بنفس الأسماء.

## النشر على GitHub Pages

1. أنشئ مستودع جديد على GitHub (مثلاً `m3dstore-nfc`).
2. ارفع كل الملفات إلى فرع `main`.
3. من إعدادات المستودع: **Settings → Pages → Source → Branch: main / (root) → Save**.
4. بعد دقيقة، الموقع يصير متاحاً على `https://<username>.github.io/<repo-name>/`.

## اختبار محلي

افتح `index.html` مباشرة في المتصفح (دبل كليك)، أو شغّل خادم محلي بسيط:

```powershell
# داخل مجلد المشروع
python -m http.server 8000
# ثم افتح: http://localhost:8000
```
